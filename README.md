# Ping

Aplicação Java com container para exemplo

## Pré-requisitos

- Java 17+
- Docker 
- Acesso a internet
- Acesso ao Docker Hub

## Instalação

#### Clone

```
git clone https://github.com/TiagoAlcan/fiap-gs1.git
```


#### Navegação

- Base

http://localhost:8080/peixes


## Features (Funcionalidades)

- Múltiplos profiles
- Banco de dados relacional
